# Модель для прогнозирования оттока клиентов для сервиса доставки кофе
Проект был выполнен в ходе обучения в Яндекс.Практикуме, профессия "Специалист по Data Science".

#### Краткое описание проекта
Необходимо провести анализ данных и построить модель бинарной классификации для прогнозирования оттока клиентов из сервиса по доставке кофе, чтобы сервис мог корректно распределять финансы при работе с клиентами.

#### Реализованные этапы проекта
- Проведен исследовательский анализ данных (EDA);
- Предобработаны данные;
- Обучена и проведена оценка базовой модели;
- Созданы и отобраны признаки;
- Подобраны гиперпараметры модели;
- Проведено финальное обучение модели и подготовлены артефактов.

#### Основной использованный в проекте стек технологий:
- **Язык**: Python
- **Обработка данных**: pandas, NumPy
- **Машинное обучение**: scikit-learn
- **Визуализация**: matplotlib, seaborn
- **Модели**: LogisticRegression
